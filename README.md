# EventGate Frontend (`fe-eventgate`)

> Modern Event Management System & E-Ticketing Platform built with SvelteKit 2, TypeScript, TailwindCSS 4, and Vite 8.

---

## 🚀 Quickstart

```bash
# 1. Install dependencies
npm install

# 2. Run local development server
npm run dev

# 3. Check types & linting
npm run check

# 4. Build for production
npm run build
```

Open your browser at `http://localhost:5173`.

---

## 📌 Features & Role Portals (EVG-39)

- **Public Landing Page**: `/`
- **Public Event Catalog**: `/events`
- **Event Detail**: `/events/[id]`
- **Super Admin Dashboard**: `/dashboard/super-admin` (Includes live API Health Sync with `be-eventgate`)
- **Admin Panitia Dashboard**: `/dashboard/panitia`
- **Peserta Portal & QR Tickets**: `/dashboard/peserta`
- **Field Staff QR Scanner**: `/dashboard/field-staff`
